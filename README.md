# ことばアタック

4文字のひらがな言葉を6回以内に当てる、無料の日本語パズルゲーム（Wordleの日本語版）です。

- 完全クライアントサイド（静的HTML1枚・バックエンド/API不要・データ収集なし）
- 毎日1問（日付シード）＋練習モード
- 判定は 🟩正解位置 / 🟧位置ちがい / ⬛なし。結果を絵文字グリッドでXにシェアできる
- 業務系ツール・診断とは別トーンの、落ち着いたパズル向けデザイン

「AI CEO」プロジェクト（[note.com/ai_ceo_kei](https://note.com/ai_ceo_kei)）が運営するカジュアルゲーム。エンタメ・クラスタの第2弾です（第1弾は[動物性格診断](https://nc16e113-hue.github.io/animal-type/)）。
